# QuiSocial
Uma simples rede social desenvolvida em React.js e Express.js

## Baixando dependências
Acesse o diretório `frontend` e execute: 
```
yarn install
```

E acesse o diretório `backend` e execute: 
```
yarn install
```

## Executando
Acesse o diretório `frontend` e execute: 
```
yarn start
```

E acesse o diretório `backend` e execute: 
```
yarn start
```
